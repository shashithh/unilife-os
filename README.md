# 🚀 UniLife OS — Smart Student Life Companion

> A centralized web application designed to help students manage academic tasks, group collaboration, wellbeing, and expenses in one platform.

---

## 🌟 Overview

UniLife OS is built to solve real-life student problems by combining productivity, teamwork, mental wellbeing, and financial tracking into a single system.

---

## 🧩 Features

* 📚 Task planning with deadlines & priorities
* 🤝 Group collaboration tracking
* 💙 Mood & stress monitoring
* 💰 Expense tracking & budgeting
* 🔔 Alerts and reminders

---

## 🏗️ Tech Stack

**Frontend**

* HTML
* CSS
* JavaScript (Vanilla JS)

**Backend**

* Node.js
* Express.js

**Database**

* MongoDB

---

## 🧭 User Flow

1. User registers and logs in
2. Dashboard displays summary
3. User navigates modules:

   * Academic Planner → manage tasks
   * Collaboration → track group work
   * Wellbeing → log mood
   * Budget → track expenses
4. System provides alerts and summaries

---

## ⚙️ Installation

```bash
git clone https://github.com/your-username/unilife-os.git
cd unilife-os
npm install
npm start
```
